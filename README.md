## Installation
***
1 Cloner le dépôt
```
git clone https://github.com/O-clock-Gaufre/projet-quizine-front.git
```
2 Aller dans le repertoire du projet
```
cd /chemin/du/projet
```
 Installer les packages NPM
```
npm install
```
